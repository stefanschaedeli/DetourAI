### New Integration Issues Found

**Issue 1: Map markers/polyline stale after route edits (UIR-06)**
- `renderGuide()` (called from all edit completion handlers) only calls `_updateMapForTab` → `fitAllStops()` (pan/zoom)
- Missing: `GoogleMaps.setGuideMarkers(plan, _onMarkerClick)` call to refresh markers
- Impact: After remove/add/reorder/replace, old numbered markers and polyline remain on persistent map
- Severity: Visual regression — content panel is correct, map panel is stale

**Issue 2: replace_stop_job not in Celery include list (CTL-04)**
- `backend/tasks/__init__.py` include list has 5 of 6 task modules — missing `tasks.replace_stop_job`
- Impact: `replace_stop_job_task.delay()` raises `celery.exceptions.NotRegistered` in production Celery mode
- Severity: Production-only failure — works in dev mode via asyncio fallback
