```python
def _get_store():
    try:
        from main import redis_client
        return redis_client
    except Exception:
        import redis as redis_lib
        return redis_lib.from_url(os.getenv("REDIS_URL", "redis://localhost:6379"))
```

Replace with a direct import -- no more deferred/lazy import needed since the circular dependency is broken:

```python
def _get_store():
    from services.redis_store import redis_client
    return redis_client
```

used throughout these files. The deferred import is still useful to avoid import-time side effects
in Celery worker processes, but the source changes from `main` to `services.redis_store`.
