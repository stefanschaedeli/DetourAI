# Phase 4 — UI Design Contract

> Visual and interaction contract for the map-centric responsive layout redesign. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | none |
| Preset | not applicable |
| Component library | none (vanilla JS, imperative DOM) |
| Icon library | inline SVG (existing pattern) |
| Font | Inter, SF Pro Display, -apple-system, sans-serif (existing `--font-primary`) |

---

## Spacing Scale

Declared values (must be multiples of 4). Uses existing CSS custom properties from `styles.css`:

| Token | Value | CSS Variable | Usage |
|-------|-------|-------------|-------|
| xs | 4px | `--space-xs` | Icon gaps, inline padding, tag pill internal spacing |
| sm | 8px | `--space-sm` | Compact element spacing, card info line gaps |
| md | 16px | `--space-md` | Default element spacing, card content padding |
| lg | 24px | `--space-lg` | Section padding, card outer gaps |
| xl | 32px | `--space-xl` | Layout gaps between map and content panels |
| 2xl | 48px | `--space-2xl` | Major section breaks, stats bar vertical margin |
| 3xl | 64px | `--space-3xl` | Page-level spacing |

Exceptions:
- Map panel gap to content panel: 0px (panels are edge-to-edge, separated by implicit boundary)
- Touch targets on mobile: minimum 44px hit area for all interactive elements (map markers, edit icons, tab buttons)

## Typography

Uses existing `--font-primary` (Inter) from `styles.css`. Phase 4 uses 4 size roles:

| Role | Size | Weight | Line Height | Usage |
|------|------|--------|-------------|-------|
| Body | 16px (`--text-base`) | 400 | 1.5 | Stop card description text, day detail content |
| Label | 14px (`--text-sm`) | 500 | 1.4 | Drive time, nights, tag pills, stat labels, overlines |
| Heading | 20px (`--text-xl`) | 600 | 1.2 | Stop card name, day heading, tab titles |
| Display | 32px (`--text-3xl`) | 700 | 1.1 | Stats bar numbers (total days, distance, budget) |

Additional constraints:
- Stop number badge on cards: 14px weight 700, white on `--accent-primary` circle
- Edit icon buttons: 20px icon size, no text label
- Map marker number badge: 14px weight 700

## Color

Uses existing CSS custom properties from `styles.css`. No new colors introduced.

| Role | Value | CSS Variable | Usage |
|------|-------|-------------|-------|
| Dominant (60%) | `#FAF8FF` | `--bg-primary` | Page background, content panel background |
| Secondary (30%) | `#FFFFFF` / `rgba(255,255,255,0.85)` | `--bg-elevated` / `--surface-card` | Stop cards, stats bar pills, day timeline cards, tab bar |
| Accent (10%) | `#7C6BE0` | `--accent-primary` | Active tab indicator, selected stop card border, active map marker, stop number badges, "Stopp hinzufuegen" CTA |
| Destructive | `#E88B8B` | `--accent-warm` | Remove stop button hover/active state, delete confirmation |

Accent reserved for:
- Active tab underline/highlight
- Selected stop card left border (3px solid)
- Map marker for currently selected stop (enlarged, accent fill)
- Stop number circle on cards
- "Stopp hier hinzufuegen?" map click prompt button
- Stats bar accent number highlights (budget remaining if positive)

Additional semantic colors:
- Route polyline on map: `#2D2B3D` (`--text-primary`, near-black) -- solid for driving, dashed for ferry (per D-11)
- Success/positive: `--accent-secondary` (`#5AB8A0`) -- budget remaining positive, completed day indicators
- Warning: `--accent-warm` (`#E88B8B`) -- budget exceeded, long drive time warning

## Layout Contract

### Desktop Split-Panel (>= 1100px)

```
+------------------------------------------+----------------------------+
|                                          |                            |
|              MAP PANEL                   |      CONTENT PANEL         |
|           (position: fixed)              |     (scrollable)           |
|              ~58% width                  |       ~42% width           |
|                                          |   +--------------------+   |
|          [numbered markers]              |   | Stats Bar (4 pills)|   |
|          [polyline route]                |   +--------------------+   |
|          [click-to-add hotspot]          |   | Tab Bar             |   |
|                                          |   +--------------------+   |
|                                          |   |                    |   |
|   +------+                               |   | Stop Card 1        |   |
|   |Sidebar|  (collapsible overlay)       |   | Stop Card 2        |   |
|   |overlay|                              |   | Stop Card 3        |   |
|   +------+                               |   | ...                |   |
|                                          |   +--------------------+   |
+------------------------------------------+----------------------------+
```

- Map panel: `position: fixed; left: 0; top: 0; width: 58%; height: 100vh;`
- Content panel: `margin-left: 58%; width: 42%; overflow-y: auto; height: 100vh;`
- Sidebar overlay: positioned absolutely over map panel, left edge, collapsible with toggle button

### Tablet (768px -- 1099px)

- Same split-panel but 50%/50% ratio
- Cards stack vertically, no horizontal row layout changes needed

### Mobile (< 768px)

```
+----------------------------+
|     MAP STRIP (~30vh)      |
|  [markers, polyline]       |
+----------------------------+
|     CONTENT (scrollable)   |
|  +----------------------+  |
|  | Stats Bar (2x2 grid) |  |
|  +----------------------+  |
|  | Tab Bar (scrollable)  |  |
|  +----------------------+  |
|  | Stop Card 1           |  |
|  | Stop Card 2           |  |
|  +----------------------+  |
+----------------------------+
```

- Map: `position: sticky; top: 0; height: 30vh;` with swipe-up gesture to minimize to ~15vh
- Content fills remaining viewport, scrolls naturally
- Stats bar: 2x2 CSS Grid instead of 4-column row
- Tab bar: horizontal scroll with `overflow-x: auto`

### Breakpoints

| Name | Value | Layout Change |
|------|-------|---------------|
| mobile | < 768px | Map strip top + full-width content below |
| tablet | 768px -- 1099px | Split panel 50/50 |
| desktop | >= 1100px | Split panel 58/42 |

## Component Contracts

### Stop Card (UIR-03)

```
+--------+-----------------------------------+
|        | #3 Annecy                    [x]  |
| 16:9   | 1h 45min vom vorherigen Stopp     |
| photo  | 2 Naechte                         |
|        | [Kultur] [See]                    |
| 120px  | Charmante Altstadt am Lac...      |
| wide   |                        [< > ...]  |
+--------+-----------------------------------+
```

- Dimensions: photo area 120px wide, 16:9 aspect ratio (120x68px), `object-fit: cover`, `border-radius: var(--radius-sm)` (8px)
- Card container: `background: var(--surface-card)`, `border: 1px solid var(--border-subtle)`, `border-radius: var(--radius)` (16px), `padding: var(--space-md)` (16px)
- Selected state: `border-left: 3px solid var(--accent-primary)`, subtle `box-shadow: 0 4px 16px rgba(124,107,224,0.12)`
- Info area: flex column, gap `var(--space-xs)` (4px) between lines
- Stop number + name: heading role (20px/600), single line with ellipsis overflow
- Drive time: label role (14px/500), `color: var(--text-secondary)`
- Nights: label role (14px/500), `color: var(--text-secondary)`
- Tags: inline flex, gap 4px. Each tag is a pill: `font-size: 12px`, `font-weight: 500`, `padding: 2px 8px`, `border-radius: 980px`, `background: var(--surface-active)`, `color: var(--accent-primary)`
- Description: body role (16px/400), `color: var(--text-secondary)`, max 2 lines with `-webkit-line-clamp: 2`
- Edit icons row: bottom-right of card, flex row, gap 8px. Icons: remove (X), reorder (grip), replace (refresh). Each 20px SVG, `color: var(--text-muted)`, hover `color: var(--accent-primary)`. Remove hover: `color: var(--accent-warm)`

### Stats Bar (UIR-05)

- Container: flex row (desktop) / 2x2 grid (mobile), gap `var(--space-sm)` (8px)
- Each stat pill: `background: var(--surface-card)`, `border: 1px solid var(--border-subtle)`, `border-radius: var(--radius-sm)` (8px), `padding: 12px 16px`
- Stat number: display role (32px/700), `color: var(--text-primary)`
- Stat label: label role (14px/500), `color: var(--text-muted)`
- Four stats: Tage (days), Stopps (stops), Distanz (km), Budget (CHF remaining)

### Day Timeline (UIR-04)

- Vertical timeline layout with left-edge line
- Timeline line: 2px solid `var(--border-default)`, left margin 20px from content edge
- Day node: 12px circle on timeline line, filled `var(--accent-secondary)` for completed, outline for upcoming
- Day card: expandable, collapsed shows "Tag N: Stop A -> Stop B, Xh Fahrt"
- Expanded day: full schedule with activities, meals, accommodation
- Collapse/expand: click day header row, 200ms slide transition with `var(--transition-default)`
- Scrollable container with all days visible in sequence

### Tab Bar

- Horizontal row, bottom-bordered
- Tabs: Uebersicht, Stopps, Tage, Kalender, Budget
- Active tab: `color: var(--accent-primary)`, 2px bottom border `var(--accent-primary)`
- Inactive tab: `color: var(--text-muted)`, no bottom border
- Tab text: label role (14px/500)
- Mobile: horizontal scroll, no wrapping

### Map Markers (UIR-06)

- Numbered circle markers using existing `GoogleMaps.createDivMarker()` pattern
- Default: 28px circle, `background: var(--text-primary)` (#2D2B3D), white number text (14px/700)
- Selected/active: 36px circle, `background: var(--accent-primary)` (#7C6BE0), white number text, `box-shadow: 0 4px 12px rgba(124,107,224,0.3)`
- Hover: scale(1.15) transition 150ms
- Route polyline: `strokeColor: '#2D2B3D'`, `strokeWeight: 3`, `strokeOpacity: 1.0`
- Ferry polyline: `strokeColor: '#2D2B3D'`, `strokeWeight: 3`, dashed pattern `[8, 6]`

### Sidebar Overlay (D-03)

- Collapsed: narrow toggle button (32px wide) on left edge of map panel, icon-only (chevron-right)
- Expanded: 240px wide panel overlaying map, `background: rgba(255,255,255,0.95)`, `backdrop-filter: blur(12px)`, left edge
- Content: existing route node visualization from sidebar.js, adapted to overlay format
- Toggle animation: 200ms slide from left, `var(--transition-default)`
- Close on click outside overlay area

### Click-to-Add Prompt (D-10)

- On map click (empty area): show small popup at click coordinates
- Popup: `background: var(--bg-elevated)`, `border-radius: var(--radius-sm)`, `padding: 12px 16px`, `box-shadow: 0 8px 24px rgba(0,0,0,0.15)`
- Content: reverse-geocoded place name (heading role, 16px/600) + "Stopp hier hinzufuegen?" button (accent pill, 14px/500)
- Dismiss: click elsewhere on map or press Escape

## Interaction Contracts

### Bidirectional Map-Content Sync (D-09)

| Trigger | Map Response | Content Response |
|---------|-------------|-----------------|
| Click map marker | Enlarge marker to selected state | Scroll to corresponding card, apply selected border |
| Click/hover stop card | Pan map to stop, enlarge marker | Apply selected card styling |
| Scroll through cards | Auto-pan map to follow reading position | N/A (user-initiated scroll) |
| Switch tab | Fit all relevant markers in view | N/A (tab content changes) |

- Auto-pan debounce: 300ms after scroll stops, only pan if the topmost visible card changed
- IntersectionObserver on each card with threshold 0.6 to detect "currently reading" card
- Smooth map pan with 400ms duration

### Drag-and-Drop Reorder

- Existing HTML5 drag-and-drop from route-builder.js, adapted for stop cards
- Drag handle: grip icon (6-dot pattern) at bottom-right of card
- During drag: card shows ghost at 50% opacity, drop zone highlighted with 2px dashed accent border
- On drop: reorder API call, optimistic UI update, metrics recalculation

### Tab Switching

- Instant content swap (no animation on content)
- Map adapts: stops tab shows all stop markers, days tab shows day segments with color coding, budget tab shows cost overlay markers
- Active tab indicator slides horizontally with 200ms transition

## Animation & Motion

All animations use existing `--transition-default: all 0.2s cubic-bezier(0.4, 0, 0.2, 1)`.

| Interaction | Animation | Duration |
|-------------|-----------|----------|
| Card selection | Border + shadow appear | 200ms |
| Map marker hover | scale(1.15) | 150ms |
| Tab switch indicator | Slide underline | 200ms |
| Sidebar expand/collapse | Slide from left | 200ms |
| Day timeline expand | Height slide + fade | 200ms |
| Map auto-pan | google.maps smooth pan | 400ms |
| Click-to-add popup appear | Fade + scale(0.95 -> 1) | 150ms |
| Card drag ghost | Opacity 0.5 | Instant |

Respects `prefers-reduced-motion: reduce` -- all transitions set to 0ms.

## Accessibility

- All map markers have `aria-label="Stopp [N]: [name]"`
- Tab bar uses `role="tablist"`, tabs use `role="tab"` with `aria-selected`
- Day timeline uses `role="list"` with expandable `role="listitem"` and `aria-expanded`
- Edit buttons have `aria-label` describing the action ("Stopp entfernen", "Stopp verschieben", "Stopp ersetzen")
- Map panel has `aria-label="Routenkarte"`, content panel has `role="main"`
- Skip-to-content link targets the content panel
- Focus management: after tab switch, focus moves to first interactive element in new tab content
- Color contrast: all text combinations meet WCAG 2.1 AA (4.5:1 body, 3:1 large text)
