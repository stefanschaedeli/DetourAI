A. Add crossfade helper _drillTransition() to guide-core.js:

Add module-level variable after existing let _guideDelegationReady = false:
  let _drillTransitionTimer = null;

Add function _drillTransition(renderFn, afterRenderFn):
  Purpose: Two-phase crossfade for drill-down navigation.
  1. Get content = document.getElementById('guide-content'). If null, return.
  2. Cancel any in-progress transition: if (_drillTransitionTimer) clearTimeout(_drillTransitionTimer) and set to null. This guards against rapid navigation (Pitfall 2 from RESEARCH.md).
  3. Phase 1 fade out: set content.style.transition = 'opacity 0.15s ease-in', content.style.opacity = '0', content.style.pointerEvents = 'none'.
  4. After 150ms timeout:
     a. Set _drillTransitionTimer = null
     b. Scroll to top (D-04): content.scrollTop = 0 AND document.getElementById('guide-content-panel').scrollTop = 0
     c. Render new content via safe DOM pattern: call renderFn() for HTML, content.textContent = '' to clear, create tmp div, insertAdjacentHTML('afterbegin', html), while (tmp.firstChild) content.appendChild(tmp.firstChild)
     d. Phase 2 fade in: set opacity '0' first, then requestAnimationFrame: set transition 'opacity 0.25s ease-out', opacity '1', pointerEvents ''
     e. Call afterRenderFn in requestAnimationFrame if provided
