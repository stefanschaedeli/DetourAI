## Component Inventory

### 1. Share Toggle Control (new)

Location: `.guide-header-row`, between the h2 title and the replan button.

**Structure:**
```
[Toggle Switch] [Label: "Teilen"] [Share URL input (readonly)] [Kopieren button]
```

**Visual spec:**
- Container: `display: flex; align-items: center; gap: 8px;` -- inline, no card/border
- Toggle switch: 40px wide x 22px tall, border-radius 11px (pill)
- Toggle unchecked: `--bg-secondary` (#F0EDF8) background
- Toggle checked: `--accent-primary` (#7C6BE0) background
- Toggle slider dot: 18px circle, white, 2px inset, `transition: transform 0.2s`
- Label "Teilen": 13px, weight 600, `--text-secondary` color
- Share URL input: appears only when toggle is ON, `--text-sm` (13px), `--bg-secondary` background, `--border-subtle` border, `--radius-sm` (8px) corners, readonly, `text-overflow: ellipsis`
- "Kopieren" button: uses existing `.btn.btn-sm.btn-secondary` styles (13px, 6px 14px padding)

**Interaction:**
- Toggle ON: POST to create share token, animate URL input appearing (fade-in 0.2s)
- Toggle OFF: show confirmation prompt (browser `confirm()` dialog), then DELETE share token
- Copy click: `navigator.clipboard.writeText(url)`, button text "Kopieren" -> "Kopiert!" for 2s

**Accessibility:**
- Toggle is `<input type="checkbox">` with `<label>` -- keyboard-accessible, screen-reader-friendly
- `aria-label="Teilen aktivieren"` on the checkbox
- URL input has `aria-label="Teilbarer Link"`

### 2. Shared View Footer (new)

Location: bottom of the guide view, below all content, only visible when `S.sharedMode === true`.

**Visual spec:**
- Full-width bar, `--bg-secondary` background
- Padding: 16px 24px
- Text: "Erstellt mit Travelman", 13px, weight 400, `--text-muted` (#9E9BB0), centered
- No link, no logo -- pure text branding per D-07

### 3. Read-Only Mode Suppressions

When `S.sharedMode === true`, the following elements are hidden (`display: none`):
- Replan button (`.replan-btn`)
- Share toggle container (`#share-toggle-container`)
- All stop edit controls (remove, add, drag handles, replace)
- Sidebar navigation
- Form/route-builder sections

No new visual elements needed -- these are CSS `display: none` rules gated on a `.shared-mode` class on `<body>` or the guide container.

## Copywriting Contract

All text in German per project convention.

| Element | Copy |
|---------|------|
| Share toggle label | "Teilen" |
| Copy button (default) | "Kopieren" |
| Copy button (feedback) | "Kopiert!" |
| Revoke confirmation | "Link deaktivieren? Bestehende Empfaenger verlieren Zugriff." |
| Shared view footer | "Erstellt mit Travelman" |
| Shared link not found (404) | "Dieser Link ist nicht mehr gueltig oder wurde deaktiviert." |
| Shared link error (network) | "Link konnte nicht geladen werden. Bitte versuche es erneut." |

Sources: D-10 (revoke confirmation), D-11 (copy feedback), D-07 (footer).

### Empty State

Not applicable -- sharing operates on existing saved trips. There is no empty state for the share feature itself. If a shared link is invalid, the 404 copy above is shown.

### Error States

| Scenario | Display |
|----------|---------|
| Invalid/revoked share token | Full-page centered message: heading "Link ungueltig", body "Dieser Link ist nicht mehr gueltig oder wurde deaktiviert.", no retry button |
| Network error loading shared trip | Full-page centered message: heading "Fehler", body "Link konnte nicht geladen werden. Bitte versuche es erneut.", retry button "Erneut versuchen" |
| Share toggle API failure | Inline toast/alert below toggle: "Teilen fehlgeschlagen. Bitte erneut versuchen." -- toggle reverts to previous state |

### Destructive Action

| Action | Confirmation |
|--------|-------------|
| Revoke share link (toggle OFF) | Browser `confirm()` dialog with copy from D-10 |

## Interaction States

### Toggle Switch States

| State | Visual |

## Layout Integration

### Guide Header Row (modified)

Current layout:
```
[h2: Ihr Reiseplan]                                    [Replan Button]
```

New layout (owner view):
```
[h2: Ihr Reiseplan]    [Toggle] [Teilen] [URL] [Kopieren]    [Replan Button]
```

The share controls sit in a flex container between the title and the replan button. On mobile (<768px), the guide header row wraps: title on first line, share controls + replan button on second line.

```css
/* Mobile wrap */
@media (max-width: 767px) {
  .guide-header-row {
    flex-wrap: wrap;
  }
  .guide-header-actions {
    width: 100%;
    justify-content: flex-start;
  }
  .share-url-input {
    max-width: 160px;  /* truncate with ellipsis on mobile */
  }
}
```
